# Capítulo 11 — Clientes Oficiales

Ejemplos de integración con los clientes oficiales de Python, Go y Java.

## Prerequisitos

- Laboratorio con perfil `novato` levantado
- Índice `productos` cargado (ver `code/ch03/`)

## Python

```bash
cd code/ch11/python
pip install -r requirements.txt
python 01_search.py
python 02_bulk.py
```

## Go

```bash
cd code/ch11/go
go run main.go
```

## Java

El archivo `SearchExample.java` es un snippet de referencia.
Requiere un proyecto Maven/Gradle con la dependencia `opensearch-java`.
